# InstaGrid Creator

InstaGrid Creator is a web-based tool that splits images into seamless grid layouts for Instagram. This tool is perfect for creating stunning and engaging Instagram profiles.

## Features

*   **Multiple Grid Formats:** Choose from various grid layouts (1x3, 2x3, 3x3) to create unique photo arrangements.
*   **Real-Time Preview:** See how your grid will look on your Instagram profile before you post.
*   **Adjustable Positioning:** Easily pan and zoom your image to get the perfect crop for your grid.
*   **Optimized for Instagram:** Images are automatically resized to Instagram's recommended dimensions (1080x1350px) for the best quality.
*   **High-Quality Output:**  The tool exports high-resolution images, ensuring your grid looks sharp and professional.
*   **Client-Side Processing:** All image processing is done in your browser, so your photos are never uploaded to a server.

## Getting Started

To get started with InstaGrid Creator, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/vivianncheng/instagrid.git
    ```
2.  **Install the dependencies:**
    ```bash
    cd instagrid
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm start
    ```
    This will open the InstaGrid Creator in your default web browser.

## How to Use

1.  **Upload Your Image:** Click the "Upload Image" button to select a photo from your computer.
2.  **Choose a Grid Format:** Select the desired grid layout from the options provided.
3.  **Adjust Your Image:** Use the pan and zoom controls to position your image within the grid.
4.  **Download Your Grid:** Once you're happy with the preview, click the "Download Grid" button to save the individual grid images to your computer.
5.  **Post to Instagram:** Upload the images to your Instagram profile in the correct order to create your seamless grid.
